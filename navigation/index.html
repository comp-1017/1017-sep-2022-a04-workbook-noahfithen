Create navigational links

Navigational links are simple anchor tags with appropriate relative addressing for the href parameter

  • page to page within the same folder
  • page to page between folders
  • stay on the current page using the pound symbol (#)

Until otherwise told, your nav semantic tag may have an optional h2 heading. Yes, you will get a document error but it will be acceptable.
